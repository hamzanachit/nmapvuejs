# NMapVueJS

A lightweight mapping library for Vue 3.

Built for people who want simple, flexible maps without dealing with heavy GIS complexity.

---

## Why this exists

Most map libraries are either too heavy or too low-level.

NMapVueJS tries to sit in the middle:
- easy like Leaflet
- modern like MapLibre
- native for Vue

---

## Installation

```bash
npm install nmapvuejs
